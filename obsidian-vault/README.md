# Obsidian Learning Vault

This vault is for structured learning notes.

## How to use
1. Open Obsidian.
2. Choose "Open folder as vault".
3. Select this folder: `obsidian-vault`.

## Suggested flow
- Quick capture in `00-Inbox`.
- Promote useful notes to `02-Learning`.
- Keep active implementation notes in `01-Projects`.
- Store external links/books/papers in `03-References`.
- Write one short daily log in `04-Daily`.
